# ChatGPT Everywhere

| ChatGPT Everywhere이란 |                             |
| ---------------------- | --------------------------- |
| `누가?`                | 모든 사람들이               |
| `언제?`                | 언제나!                     |
| `어디서?`              | 크로미움 사이트를 쓸 때     |
| `무엇을?`              | GPT AI에 질문할 수 있게     |
| `왜?`                  | 항상 들어가기 귀찮으니깐!   |
| `어떻게?`              | 브라우저 확장 프로그램으로! |

## How to test

```bash
npm install
npm run dev
```

```bash
pnpm i
pnpm run dev
```
